# Delivery-Time-Prediction

# Project Overview

This project aims to predict the delivery time (in minutes) based on various factors such as distance, traffic conditions, weather, and time of day.

The model helps in improving logistics efficiency and providing accurate delivery estimates.

# Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## ⚙️ Feature Engineering

A new feature was created to improve model performance:

Speed Feature

Speed = Distance / Delivery Time

This helps the model better understand delivery efficiency.

# Model Performance

Evaluation metrics used:

* R² Score
* Mean Absolute Error (MAE)
